# Nothing But The Truth: Educational Platform

A full-stack, data-driven Single Page Application (SPA) designed for the structured delivery of high-volume educational and theological content.

## 🚀 Architectural Overview
This platform utilizes a **JSON-driven architecture** to separate content from logic, allowing for rapid updates and high scalability.

### 🛠 Core Features & Key Modules
- **Dynamic Content Engine:** Architected for JSON-driven delivery of 23+ individual studies and 19+ daily devotionals, minimizing frontend hard-coding.
- **Gamification Engine:** Integrated 20+ interactive quiz and game modules featuring persistent state logic to enhance user retention.
- **Advanced Media Management:** Managed 16+ custom video assets utilizing localized CSS/JS players for optimized delivery.
- **Modular Architecture:** Engineered with a clean separation of concern between Content (JSON), Logic (JS), and Presentation (CSS).
- **Scalable Data Layer:** Implemented relational JSON structures to allow for rapid data updates and schema consistency.
- **User Engagement Logic:** Designed frameworks for progress tracking and personalized user experiences within the SPA.

## 💻 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Data Layer:** Relational JSON structures.
- **Environment:** Developed and tested within **Visual Studio Code**.

## 📊 Current Project Scope
- **Total Modules:** 60+ interactive components.
- **Development Philosophy:** Priority on modularity, high-precision data mapping, and industrial-grade troubleshooting logic.
