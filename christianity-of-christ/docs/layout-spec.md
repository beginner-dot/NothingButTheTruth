# Figma-Style Layout Notes

## Dashboard Page
- Frame: 1440px desktop, 768px tablet, 390px mobile.
- Header row: page title + subtitle.
- Left panel (220px): Progress summary with circular ring and weekly notice.
- Main panel: Habit tracker cards in 2-column desktop / 1-column mobile.
- Section below: Reality Check card with top-3 struggle list and recommendations.
- Footer sections: Encouragement feed, Reflection editor, Data controls.
- Spacing: 24px section gaps desktop, 16px mobile.

## Lesson Page
- Hero card: lesson title, module ID, estimated minutes.
- Content body card: MDX prose with clear heading hierarchy and blockquote styles.
- Quiz card: below lesson content.
- Navigation: top back link to course home.
- Accessibility: keyboard focus preserved, high contrast toggle in global header.
